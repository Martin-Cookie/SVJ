Koordinace a workflow
koordinace Orchestrátor
ORCHESTRATOR.md
Zeptá se co potřebuješ, zanalyzuje stav, navrhne plán agentů s módem (rychlý/hluboký), spouští je postupně s předáváním kontextu. Sám nic neopravuje.
Přečti ORCHESTRATOR.md a zkoordinuj údržbu projektu.
koordinace Session Start
SESSION-START.md
Orientace na začátku session — git stav, reporty, co dál.
Přečti SESSION-START.md a proveď orientaci v projektu.
automatický Task Agent
CLAUDE.md
Automaticky při každém úkolu — plán, mockupy, schválení, potvrzení commitu, report.
Stačí napsat úkol — pravidla se aplikují automaticky.
Analytičtí agenti (nic neopravují, pouze reportují)
analytický Code Guardian
~4/6 min → AUDIT-REPORT.md
Audit kódu, bezpečnosti, výkonu, error handlingu, git hygieny, testů.
Přečti CODE-GUARDIAN.md a proveď audit projektu. Výstupem je AUDIT-REPORT.md. Nic neopravuj, pouze reportuj.
analytický Test Agent
~5/12 min → TEST-REPORT.md
Pytest, route coverage, Playwright smoke + funkční testy, exporty, back URL.
Přečti TEST-AGENT.md a proveď kompletní testování projektu. Výstupem je TEST-REPORT.md. Nic neopravuj, pouze testuj a reportuj.
analytický UX Optimizer
~4/8 min → UX-REPORT.md
Analýza UX z 6 pohledů: uživatel, business, designer, performance, error recovery, data quality.
Přečti UX-OPTIMIZER.md a analyzuj celou aplikaci. Výstupem je UX-REPORT.md. Nic neopravuj.
analytický Doc Sync
~5/10 min → opravy v docs
Synchronizace dokumentace s kódem — zastaralé, chybějící, duplikáty, křížové odkazy.
Přečti DOC-SYNC.md a proveď synchronizaci dokumentace s aktuálním stavem projektu.
analytický Business Logic
~8 min → BUSINESS-LOGIC.md
Extrakce business procesů, pravidel, datového modelu, edge cases, integrací.
Přečti BUSINESS-LOGIC-AGENT.md a proveď extrakci business logiky. Nic neopravuj.
analytický Purge/Restore/Verify
~1 min → purge report
Automatický end-to-end test: záloha → purge → restore → ověření. Nahrazuje Backup Agent.
Přečti PURGE-RESTORE-VERIFY.md a proveď end-to-end test zálohovacího systému.
Akční agenti (provádějí změny)
akční Release Agent
~5 min
Pre-release kontrola, changelog, git tag, ZIP balíček.
Přečti RELEASE-AGENT.md a připrav release. Nejdřív proveď pre-release kontrolu, pak po schválení připrav balíček.
akční USB Deploy
~25 min
Přenos na jiný Mac: příprava zdrojového Macu, setup cílového, verifikace, troubleshooting.
Přečti docs/agents/USB-DEPLOY.md a proveď přenos aplikace na USB. Cíl: /Volumes/NAZEV_USB
akční Cloud Deploy
~5 min
Analýza připravenosti pro cloud, doporučení platformy (VPS/PaaS/Docker), příprava nasazení.
Přečti CLOUD-DEPLOY.md a analyzuj připravenost projektu pro nasazení do cloudu. Navrhni nejlepší variantu.
Doporučené scénáře
Po bloku změn Code Guardian → Doc Sync → Test Agent (rychlý)
Před releasem Code Guardian → P/R/V → Doc Sync → Test Agent → Release (hluboký)
Zlepšení UX UX Optimizer → Doc Sync
Dokumentace Business Logic → Doc Sync
Kompletní údržba CG → Doc → Test → UX → P/R/V → BL
Nevím co dělat Spustit Orchestrátora — zeptá se a navrhne plán
